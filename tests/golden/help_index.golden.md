# Subcog Help

Welcome to Subcog, the persistent memory system for AI coding assistants.

## Available Topics

- **[Getting Started with Subcog](subcog://help/setup)**: Installation and initial configuration guide
- **[Core Concepts](subcog://help/concepts)**: Understanding namespaces, domains, URNs, and memory lifecycle
- **[Capturing Memories](subcog://help/capture)**: How to capture and store memories effectively
- **[Searching Memories](subcog://help/search)**: Using hybrid search to find relevant memories
- **[Integration Workflows](subcog://help/workflows)**: Hooks, MCP server, and IDE integration
- **[Troubleshooting](subcog://help/troubleshooting)**: Common issues and solutions
- **[Advanced Features](subcog://help/advanced)**: LLM integration, consolidation, and optimization
- **[User-Defined Prompts](subcog://help/prompts)**: Save, manage, and run prompt templates with variables

## Quick Start (MCP Tools)

1. **Capture**: Use `subcog_capture` tool with `namespace` and `content`
2. **Search**: Use `subcog_recall` tool with `query` parameter
3. **Status**: Use `subcog_status` tool
4. **Browse**: Use `subcog://project/_` resource
